## 5. AI Hub 전체 수치와의 차이

비교:

- AI Hub: 이미지 `13,936`, 라벨 `125,424`
- 로컬: 이미지 `12,545`, 라벨 `112,905`

차이:

- 이미지 `1,391장` 부족
- 라벨 `12,519개` 부족

해석:

- 로컬 자료는 전체 공개본과 일치하지 않음
- AI Hub 페이지의 `샘플(경량) 데이터는 원본과 차이가 있을 수 있음` 안내와 합치되므로, 현재 폴더는 부분/경량 버전일 가능성이 높음

이 해석은 AI Hub 안내 문구와 로컬 전수 집계 차이를 바탕으로 한 추론임.

## 6. 모델 코드 점검

원본 공개 소스는 도커/리눅스 중심으로 작성되어 있었음.

로컬 실행에 직접 영향을 주던 문제:

- 윈도우 경로에서 `folder_path.split("/")` 사용
- `cv2.imread()` 가 한글 디렉토리에서 실패
- `torch.load()` 에 `map_location` 없음
- 좌우 대칭 영역(`4`, `6`) 뒤집기 비교가 문자열/정수 불일치로 무효화
- `test.py` 가 `date`, `ls -al` 같은 리눅스 명령 호출
- 기본 ResNet 가중치를 온라인 다운로드하려는 구조

이 문제들을 피하기 위해 `model/runtime/evaluate_classification.py` 기반의 윈도우 실행 경로를 별도로 구성함.
